Это минимальный срез того что можно сделать с FreeBSD сразу после установки. Всё это есть в FreeBSD Handbook, в wiki.freebsd.org, в man, но тут всё какбэ на одной страничке. ===== Качать файлы можно с помощью fetch, например fetch --no-verify-peer https://vall.su/raw/freebsd_postinstall.4.txt Опция --no-verify-peer отключает проверку сертификата SSL, потому что FreeBSD в базовой системе поставляется без списка корневых сертификатов, необходимых для работы с https. Пакет с корневыми сертификатами можно поставить так # pkg install ca_root_nss ===== Если поставил MS Windows после FreeBSD на смежный раздел, то чтобы восстановить загрузчик FreeBSD, можно загрузиться с liveUSB и сделать # boot0cfg -B ada0 Имя диска (ada0) можно узнать из $ dmesg | grep ada ===== Поменять shell для root с tcsh на sh # pw user mod -n root -s sh ===== adduser -- диалог для создания пользователя Login group: wheel -- чтоб иметь возможность делать su Invite into other groups: operator -- чтоб иметь возможность выключать комп Чтоб создался пользователь без пароля нужно отвечать так Use password-based authentication? -- yes Use an empty password? -- yes Очистить пароль пользователя # pw user mod -n username -w none Добавить пользователя в группу operator # pw group mod operator -m username ===== Включить kms-драйверы для видеокарты временно # kldload i915kms -- для Intel или # kldload radeonkms -- для Radeon Включение при загрузке -- в /etc/rc.conf kld_list="i915kms" или kld_list="radeonkms" Начиная с FreeBSD 13 KMS-драйверы не входят в базовую систему и должны ставиться из мета-порта drm-kmod и прописываться в /etc/rc.conf так kld_list="/boot/modules/i915kms.ko" или kld_list="/boot/modules/radeonkms.ko" ===== Если загружается vt в текстовом режиме (в VirtualBox) -- в /boot/loader.conf hw.vga.textmode=0 Если надо загрузить syscons вместо vt -- туда же kern.vty=sc Загрузить точно vt kern.vty=vt ===== Отключить вывод системных сообщений в консоль -- в /etc/syslog.conf закомментировать строку # *.err;kern.warning;auth.notice;mail.crit /dev/console и перезагрузить syslogd # service syslogd restart ===== /etc/wall_cmos_clock -- наличие этого пустого файла сообщает системе, что BIOS настроен на местное (фактическое) время; отсутствие же этого файла означает, что время в BIOS (CMOS) настроено на UTC, а время в системе рассчитывается по часовому поясу. Подробнее: man adjkerntz ===== Настройка времени ожидания загрузчика до начала загрузки -- в /boot/loader.conf autoboot_delay="3" -- для трёх секунд, например. ===== Для проводного интернета в /etc/rc.conf ifconfig_re0="DHCP" -- где 're0' -- имя карты в ifconfig ===== Для WiFi # sysctl net.wlan.devices -- так можно глянуть имя wifi-карты, например 'ath0' И прописать её в rc.conf wlans_ath0="wlan0" ifconfig_wlan0="WPA SYNCDHCP" Для пароля # wpa_passphrase вайфай пароль > /etc/wpa_supplicant.conf И затестить # service netif restart ===== Synaptics touchpad /boot/loader.conf hw.psm.synaptics_support="1" /etc/rc.conf moused_enable="YES" ===== Чтобы при входе в систему не показывалось сообщение "Welcome to FreeBSD!" (motd), можно сделать $ touch ~/.hushlogin или отредактировать класс default в /etc/login.conf (если пользователь в этом классе или в дочернем), добавив :hushlogin: после чего # cap_mkdb /etc/login.conf ===== Для русской локализации в консоли можно добавить пользователя в класс 'russian' # pw user mod user -L russian Все классы можно посмотреть в /etc/login.conf Другой путь -- задать переменную LANG, которую можно прописать в ~/.profile $ export LANG=ru_RU.UTF-8 Файлы локализации находятся в /usr/share/locale/ Для русского ввода в терминале нужно добавить это в /etc/rc.conf keymap="ru.kbd" -- для переключения по CapsLock keymap="ru.win.kbd" -- для переключения по Ctrl+Shift Файлы клавиатур находятся в /usr/share/vt(syscons)/keymaps/ Также файл клавиатуры можно применить временно: $ kbdcontrol -l /usr/share/vt/keymaps/ru.win.kbd ===== Если перехимичил с шрифтами, то вернуть шрифт по-умолчанию (Terminus 12 кегль) можно так: $ vidcontrol -f Так же через vidcontrol или vidfont можно поменять или примерить используемый шрифт. ===== Обновить систему # freebsd-update fetch # freebsd-update install ===== Установка маломальского X11 # pkg install xorg-server xinit xterm setxkbmap xset xrdb font-misc-misc \ xf86-input-keyboard xf86-input-mouse xf86-video-intel fluxbox Конечно, вместо xf86-video-intel надо ставить соответствующий драйвер видеокарты xf86-video-ati -- для карт Radeon, или если старая видеокарта, такая как X1600, то xf86-video-ati-legacy -- драйвер для старых карт.
