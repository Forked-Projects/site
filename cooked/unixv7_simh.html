Запуск Unix v7 в эмуляторе SIMH Скачаем сам образ диска Research Unix 7 http://simh.trailing-edge.com/kits/uv7swre.zip в нём unix_v7_rl.dsk -- это образ диска. Сборки SIMH для Windows https://github.com/simh/Win32-Development-Binaries в архиве интересует файл pdp11.exe -- это эмулятор PDP-11. Для Unix-like системы установите simh из репозитория, например для FreeBSD: # pkg install simh Распакуем в каталог, например в "unixv7", образ unix_v7_rl.dsk и pdp11.exe. Перейдём в каталог unixv7, в нём запускаем эмулятор: $ pdp11 Для rpm-based дистрибутивов, похоже, исполняемые файлы SIMH называются несколько по-другому: $ simh-pdp11 sim> -- это приглашение командной строки эмулятора. Настраиваем систему, передавая команды терминалу simh: sim> set cpu 11/45 -- определяет тип эмулируемой системы, Disabling XQ sim> set tto 7b -- устанавливает разрядность эмулируемого телетайпа в 7 бит, sim> att rl unix_v7_rl.dsk -- монтирует образ диска к контроллеру rl, sim> boot rl -- загружает наш образ. @ Для автоматизации запуска эмулятора pdp11 можно создать файл pdp11.ini в каталоге unixv7 с вышеприведёнными командами: $ cat pdp11.ini set cpu 11/45 set tto 7b att rl unix_v7_rl.dsk boot rl $ Далее, в приглашении @ вводим: @boot New Boot, known devices are hp ht rk rl rp tm vt : Указываем раздел (0,0) и образ ядра(rl2unix), который необходимо загрузить: : rl(0,0)rl2unix mem = 177856 # И сразу попадаем в командную строку под суперпользователем. # ls -l total 642 drwxrwxr-x 2 bin 2512 Sep 22 05:32 bin -rwxr-xr-x 1 bin 8986 Jun 8 1979 boot drwxrwxr-x 2 bin 160 Sep 22 05:47 dev drwxrwxr-x 2 bin 336 Sep 22 05:33 etc -rwxr-xr-x 1 sys 53302 Jun 8 1979 hphtunix -rwxr-xr-x 1 sys 52850 Jun 8 1979 hptmunix drwxrwxr-x 2 bin 320 Sep 22 05:33 lib drwxrwxr-x 2 root 96 Sep 22 05:46 mdec -rwxr-xr-x 1 root 50990 Jun 8 1979 rkunix -rwxr-xr-x 1 root 51982 Jun 8 1979 rl2unix -rwxr-xr-x 1 sys 51790 Jun 8 1979 rphtunix -rwxr-xr-x 1 sys 51274 Jun 8 1979 rptmunix drwxrwxr-x11 root 176 Sep 22 05:46 usr # Замечание: для удаления последнего введённого символа в Unix используется символ # , а не клавиша Backspace.
