If you want russian tty you can add to /etc/rc.conf
keymap="ru.kbd" -- for switching between eng and ru keys with CapsLock
keymap="ru.win.kbd" -- for switching with Ctrl+Shift

And add your user to class 'russian'
pw user mod user -L russian

Also for forcing load VT add to /boot/loader.conf
hw.vga.textmode=0

Or for cool resolution add to /boot/loader.conf
kldload i915kms
or
kldload radeonkms

=====

For wired internet add to /etc/rc.conf
ifconfig_re0="DHCP"

For WiFi
sysctl -a | grep wlan  -- it gives 'name0' name of card

In rc.conf
wlans_name0="wlan0"
ifconfig_wlan0="WPA SYNCDHCP"

And for storing pass
wpa_passphrase ssid pass > /etc/wpa_supplicant.conf

=====

Change font to vgafont in rc.conf
font8x16="/usr/share/vt/fonts/vgarom-8x16.fnt"
