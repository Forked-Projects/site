Нужно подключить драйверы для видеокарты в /boot/loader.conf
или исполнить временно
kldload i915kms -- для Intel
или
kldload radeonkms -- для Radeon

Также если загружается syscons вместо vt -- в /boot/loader.conf
hw.vga.textmode=0

Изменить шрифт на олдскульный vgafont в rc.conf
font8x16="/usr/share/vt/fonts/vgarom-8x16.fnt"
или временно
vidcontrol -f /usr/share/vt/fonts/vgarom-8x16.fnt

Для русского ввода в терминале нужно добавить это в /etc/rc.conf
keymap="ru.kbd" -- для переключения по CapsLock
keymap="ru.win.kbd" -- для переключения по Ctrl+Shift

И добавить пользователя в класс 'russian'
pw user mod user -L russian

=====

Для проводного интернета в /etc/rc.conf
ifconfig_re0="DHCP" -- где 're0' -- имя карты в ifconfig

For WiFi
sysctl -a | grep net.wlan.devices  -- так можно глянуть имя wifi-карты 'name0'

И прописать её в rc.conf
wlans_name0="wlan0"
ifconfig_wlan0="WPA SYNCDHCP"

Для пароля
wpa_passphrase вайфай пароль > /etc/wpa_supplicant.conf
